# Attendance System

A web-based attendance management system built with Node.js, Express, MongoDB, and JavaScript.

---

## **Requirements / Prerequisites**

Before running the project, ensure the following are installed on your system:

1. **Node.js**  
   - Version 18 or higher recommended.  
   - [Download Node.js](https://nodejs.org/)

2. **npm (Node Package Manager)**  
   - Comes bundled with Node.js.  

3. **MongoDB**  
   - Either install locally: [MongoDB Community Server](https://www.mongodb.com/try/download/community)  
   - Or use MongoDB Atlas for a cloud database: [MongoDB Atlas](https://www.mongodb.com/cloud/atlas)


4. **Frontend Browser**  
   - Any modern browser (Chrome, Firefox, Edge).
